# Mini Smart Home System

## Project Description
This project is an Arduino-based mini smart home system developed in Tinkercad.

The system uses:
- light sensor (LDR)
- temperature sensor (TMP36)

Based on the measured values, the Arduino automatically controls:
- lighting
- ventilation

The system also contains:
- LCD display
- manual light control button

## Features
- Automatic lighting control
- Temperature-based fan control
- Real-time LCD display
- Manual override button

## Components
- Arduino UNO
- LCD 16x2
- LDR sensor
- TMP36 temperature sensor
- LEDs
- Push button
- Resistors
- Breadboard

## Software
- Arduino IDE / Tinkercad
- C++
